  <h2>주요기능</h2>
  
### 회원 (User)
FireBase/회원가입 / 로그인/ 로그아웃/FireStore/피드 및 스터디 등록 / 마이페이지 사진 등록 / 그룹 참여 
  
### 실시간 채팅 (Chat)
실시간으로 다른 유저와 채팅을 할 수 있는 기능 구현 /
아이디 등록시 만든 닉네임 or 사용 할 닉네임으로 채팅 참여 가능

### 디테일 페이지 (detail)
스터디 및 챌린지 참여 기능 및 찜 기능 구현
